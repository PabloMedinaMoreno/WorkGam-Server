## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language**: JavaScript
- **Package Manager**: npm

### Installation

Build **WorkGam-Server** from the source and install dependencies:

1. Clone the repository:
    ```bash
    git clone https://github.com/PabloMedinaMoreno/WorkGam-Server
    ```

2. Navigate to the project directory:
    ```bash
    cd WorkGam-Server
    ```

3. Install the dependencies:
    ```bash
    npm install
    ```

### Usage

Run the project with:

```bash
npm start
```

### Testing

To execute the test suite, use:

```bash
npm test
```
